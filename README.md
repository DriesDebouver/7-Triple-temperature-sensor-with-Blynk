# Triple Temperature Sensor With Blynk App
Triple temperature sensors measures the temperature and displays them on a OLED display and in the Blynk smartphone app.
Based on ESP32.

- 2x Dallas DS18B20
- 1x KTY81-220
- 1x SSD1306 OLED Display
- 1x 16bit ADC
